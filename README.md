# ICAs
Different Iterative Collective Classifiers - Multi-label | Multi-Relational | Ordered labeling

1 MLCC - Multi-Label Collective Classification, SDM'11 <\br>
2 CEC - Across-model collective ensemble classification, AAAI'11 <\br>
3 ICA - Iterative Classification Algorithm <\br>
4 Ordered - Nodes are labeled in ICA ordered by fraction of labeled immediate neighbors 

Requires Libsvm
